# ¿Cómo subir un module a NPM?

## Primero, tenemos que tener una cuenta en NPM

```bash
# Crear cuenta por consola
npm adduser
```

### Comandos de interes

```bash
# Para cambiar las versiones
npm version 1.0.1

# o
yarn version
```

### Publicar módulo

```bash
# Para cambiar las versiones
npm publish

# o
yarn publish
```
